Editeur de map du jeu.

Utilisation: le lancer, pour charger un map, la placer dans le même repertoire que "map_editor.pde" puis appuyer sur "load",
toute autre procédure entrainera une erreur de type "NullPointerException"

Une fois la map créée, cliquer sur le boutton "save". ATTENTION : On ne peut sauvegarder QU'UNE SEULE FOIS, une fois la map enregistrée,
il faut ensuite relancer le programme. La map créée sera nommée "map.txt" et déposée au même emplacement que le fichier "map_editor.pde"
